# PowerShell
1.  PowerShell is cross platform task automation solution.
2.  PowerShell tasks can be performed by using commands or scripting.
3.  PowerShell runs on Linux, Windows, macOS.
4.  PowerShell modules can be used in Azure, AWS, Windows, SQL, VMWare, Google Cloud etc.
5.  PowerShell enables you to manage your enterprise infrastructure with configuration as code
